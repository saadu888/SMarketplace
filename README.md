# Temaka App

Social media app for community with marketplace features.

## Features
- User registration and login
- Social feed with posts, likes, and comments
- Groups creation and management
- Marketplace for buying and selling

## How to Run
This is a React Native app. To run it:

1. Install Expo Go on your phone
2. Scan the QR code from Expo Snack
3. Or run locally with: `npx expo start`

## Technologies
- React Native
- Expo
- JavaScript
